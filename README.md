# Ecommerce

Sistema de e-commerce desenvolvido com:

- Node.js
- Express
- Handlebars
- MySQL

## Instalação

```bash
npm install
```

## Configuração

Crie um arquivo `.env`

Exemplo:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=senha
DB_NAME=ecommerce
PORT=3000
```

## Executar

```bash
npm run dev
```
